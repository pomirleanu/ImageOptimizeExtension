# Azure PngQuant and JpegOptim Setup

Experimental SiteExtension for Azure with pngquant and jpegoptim release and setup for azure web app.



### Installation

* Set your Azure site up to Continuos deployment
* Install the [Imagick ](https://www.siteextensions.net/packages/ImageOptimizationExtension/) extension from the extension manager
